# Machine Learning中的基本概念

* Supervised Learning：监督学习
* Unsupervised Learning：无监督学习
* Hypothesis:假设
* Regression:回归
* Local Regression:局部回归
* Overfitting:过拟合
* Classification：分类
* Independently and Identically：独立同分布
* Feature values: 特征值
* Variance：方差，模型过于复杂导致不确定性
* Bias:偏差，模型过简单导致的不准确
* Discrete：分离的，不连续的，分散的
* Continuous：连续的
* Decuction：演绎
* Induction：归纳
